DOCTOR BOOKING APP - SETUP INSTRUCTIONS

This ZIP contains all files for your Doctor Booking Application with:
✅ 12 Doctors with Serial Numbers
✅ Patient Booking Chatbot
✅ Admin Dashboard
✅ Send Reminders to Patients
✅ WhatsApp Integration

FILES INCLUDED:
1. index.html - Main HTML file
2. styles.css - All CSS styling
3. script.js - JavaScript with all functionality

QUICK START:
1. Extract all files to a folder
2. Open in VS Code or any text editor
3. Right-click index.html -> "Open with Live Server"
4. App opens at http://localhost:5500

DEPLOYMENT:
- Netlify: Drag & drop folder at netlify.com
- GitHub Pages: Push to GitHub and enable Pages
- Vercel: Import repository and deploy
- AWS S3: Upload files to S3 bucket

FEATURES:
- 12 Real Doctors
- Serial Number System (#001, #002, etc.)
- Patient Name & Phone Collection
- Admin Dashboard with 4 Stats Cards
- Send Reminders to Patients
- WhatsApp Direct Links
- Responsive Design
- No Database Required

Enjoy your appointment booking app!
